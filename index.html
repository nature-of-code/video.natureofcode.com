Welcome

This site organizes a series of video lessons for my Nature of Code book. I also have a beginner programming series.

Help

This is an experiment and there's lots of room for improvement. If you'd like to learn more about how these lessons were made, you can read this post and watch this video.

If you'd like to report a bug in this site or a technical problem with one of the videos, you can file a github issue. This site is a work in progress and we will also gladly accept contributions.

Questions

If you have questions about the video content and examples, please use the comments on each page. I am curious as to whether this is a useful format for queries and discussion around the content.

Annotations

You'll notice under each video there is an annotation box.

This is what an annotation looks like. It might have a link in it.

The annotations are timecoded and appear for a block of time in the video. If you would like to help contribute to the annotations, there is more info on github.
